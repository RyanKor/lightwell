TIG 500EP is a DC TIG welder by digital control method for various pulse function and manual welding is possible; P.W.M current control inverter method is used with IGBT element and it has the advantages of small size, light weight, and power saving.

Input Electricity

Both 220V/380V, 50/60Hz

Output Current

TIG 10~500A, MMA20~400A

Output Load Voltage

TIG 20~230V, MMA25~40V

No-Load Voltage

68V

Slope Time

Moving up for 0~10sec and moving down for 0~15sec

Gas Flow Time

Start 0sec and later 0~15sec

Pulse Frequency

0.5~200Hz

Pulse Bandwidth

5~85%

Rate of Use

300A 50% (25° outside)

Dimensions

320 x 600 x 560

Weight (Net)

(NET)47kg
 

  • Digital Control and Automatic Setting Function – As the set data may be automatically saved with application of Digital Control, the operation is simple.
  • As Start Current Control Function for Sustainable Arc Start is provided, perfect arc start is possible and the welder shows excellent efficiency in case of continuous tag welding.
  • Various Welding Functions – One welder has three functions: common DC TIG welding (2T), DC TIG welding (4T) using holding function, and manual welding (DC Arc, MMA).
  • Hot Start Current Control – This function supports smooth arc start.
  • Start Current Control Function – The arc strength at the start of welding is controlled to meet the requirement related with thin plate crack and high-speed start for rapid tack welding.
